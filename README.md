# PtSe2-MoTe2-bipolar-pulses
MCP
The code is divided into two main parts corresponding to the functionalities described in the paper:

Convolution_Program/: Contains the MATLAB scripts for Multispectral Convolution Processing (MCP). This program uses the hardware-measured kernel weights to perform image convolution (e.g., Gaussian blur, sharpening).

Encryption_Program/: Contains the MATLAB scripts for cryptographic key generation and the AES-inspired custom cipher for image encryption/decryption.

Data/: Contains sample kernel matrix data required to run the above programs.

🚀 How to Run
Open MATLAB and set this repository as your Current Folder.

To verify the convolution processing, navigate to the Convolution_Program folder and run the main script.

To verify the encryption/decryption process, navigate to the Encryption_Program folder and run the main script.
